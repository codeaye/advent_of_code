## Run Locally

Clone the project

```bash
  git clone https://github.com/codeaye/advent_of_code
```

Go to the project directory

```bash
  cd advent_of_code
```

Then run the solutions for the day you want, ie:
```bash
cargo run --release --bin day_1
```
